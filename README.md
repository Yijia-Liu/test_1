# VisCA For CCS_2026

1. We provide an example of QQP-2 task with LLM of gpt3.5.
   
3. The adversarial samples generated by VisCA are included in:
   - question1_replace.txt
   - question2_replace.txt
   
   Or run 1.py to get a new one.

4. Precomputed model predictions are provided in the answer.txt

5. To reproduce the reported attack success rate (ASR), run:
   com.py

Note:

To avoid dependencies on external APIs and ensure reproducibility, we provide precomputed outputs instead of requiring API access. The full pipeline will be released upon acceptance.
