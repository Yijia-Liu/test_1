# VisCA For CCS_2026

1. Navigate to the provided dataset directory (e.g., QQP-2-test).
   
3. The adversarial samples generated by VisCA are included in:
   - question1_replace.txt
   - question2_replace.txt

4. Precomputed model predictions are provided in the results/ directory.

5. To reproduce the reported attack success rate (ASR), run:
   python eval.py

Note:
To avoid dependencies on external APIs and ensure reproducibility, we provide precomputed outputs instead of requiring API access. The full pipeline (including model querying) will be released upon acceptance.
